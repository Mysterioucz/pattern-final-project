# Plan 02-02: Implement KD Loss and Training Utilities

**Objective**: Implement the Knowledge Distillation loss function using temperature-scaled softmax and KL-Divergence.

## Tasks

<id>02-02-01</id>
<objective>Implement Knowledge Distillation Loss</objective>
<read_first>
- .planning/phases/02-model-implementation/02-CONTEXT.md
- projects/src/config.py
</read_first>
<action>
Create `projects/src/utils/losses.py` and implement `distillation_loss(student_logits, teacher_logits, labels, alpha, temperature)`.
The function must:
1. Compute the supervised Cross-Entropy loss ($L_S$).
2. Compute the KL-Divergence loss ($L_D$) on soft targets:
   - Apply `F.log_softmax(student_logits / temperature, dim=1)`
   - Apply `F.softmax(teacher_logits / temperature, dim=1)`
   - Use `F.kl_div` with `reduction='batchmean'`.
3. Scale $L_D$ by $temperature^2$.
4. Return `alpha * L_S + (1 - alpha) * L_D`.
</action>
<acceptance_criteria>
- `projects/src/utils/losses.py` exists.
- Contains scaling by $T^2$ (e.g., `* (temperature ** 2)`).
- Uses `F.kl_div`.
</acceptance_criteria>
</task>
