# Milestone v1.0: Roadmap Archive

**Date:** 2026-05-06
**Version:** v1.0

## Overview
Modularizing an existing GCN-based Bitcoin fraud detection notebook into a maintainable codebase and implementing Knowledge Distillation to compress the model into an efficient student version.

## Phases

- [x] **Phase 1: Infrastructure & Modularization** - Extract logic into lean `.py` files and set up data pipeline.
- [x] **Phase 2: Model Implementation** - Define Teacher and Student GCN architectures and KD loss.
- [x] **Phase 3: Training & Distillation** - Train Teacher baseline, Vanilla Student, and Distilled Student.
- [x] **Phase 4: Evaluation & Benchmarking** - Compare performance and efficiency across models.

1. Modularized existing notebook implementation into `projects/src/`.
